## 🐳 Docker: Первое знакомство

Репозиторий для первого погружения в мир Docker. Здесь мы разберём базовые команды, запустим контейнер и посмотрим, как всё устроено изнутри.

> ⚠️ **Важно:** Перед стартом проверь, что порт `8088` свободен. Если нет — останови приложение, которое его использует.

---

### 🌐 Порт 8088 открыт

Перед запуском контейнера порт доступен и готов к работе.

![Порт свободен](https://ibb.co/yc2qk61M)

---

### 🚀 Запуск контейнера

Скачиваем официальный образ и запускаем контейнер с пробросом порта:

```bash
docker run -d -p 8088:80 --name welcome-to-docker docker/welcome-to-docker
```

После выполнения команды контейнер стартует в фоновом режиме.

![Контейнер запущен](https://ibb.co/mCSnKXvk)

---

### 👀 Что внутри?

Открываем браузер и переходим по адресу:  
👉 [http://localhost:8088](http://localhost:8088)

Видим приветственную страницу Docker.

![Страница в браузере](https://ibb.co/4RDpDwrQ)

---

### 🖥️ Подключаемся к контейнеру

Заходим внутрь запущенного контейнера, чтобы изучить его "изнутри":

```bash
docker exec -it welcome-to-docker /bin/sh
```

---

### 🔍 Изучаем систему внутри контейнера

Проверяем версию ядра и запускаем мониторинг процессов:

```bash
uname -a
top
```

![Информация о системе](https://ibb.co/Jj2kvS7Z)

---

### 📦 Обновление пакетов

Внутри контейнера — минималистичный Alpine Linux. Обновляем список пакетов и сами пакеты:

```bash
apk update && apk upgrade
```

![Процесс обновления](https://ibb.co/fYD5rfVV)

---

### ✨ Красивый вывод информации

Устанавливаем утилиту `fastfetch` для отображения информации о системе в стиле "neofetch":

```bash
apk add fastfetch
fastfetch
```

![fastfetch в контейнере](https://ibb.co/Z60Chhz0)

---

### 🔙 Возвращаемся в основную систему

Выходим из контейнера обратно в терминал хоста:

```bash
exit
```
